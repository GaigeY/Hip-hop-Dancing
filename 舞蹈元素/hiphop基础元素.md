# Hiphop Old School元素表

> 写在前面：元素只是最最基础的动作原型，元素之间的混合、方向高度的变化、发力方式的改变、律动与节奏的结合、轨迹的变型都可以组合出无数的可能。

| 序号 | 名称                         | 备注                             | 来源类型    |
| ---- | ---------------------------- | -------------------------------- | ----------- |
| 1    | Bounce                       | up-down                          |             |
| 2    | Rock                         | 前后摇动                         |             |
| 3    | Skate                        | 滑步                             |             |
| 4    | Roll                         | 画圆                             |             |
| 5    | Wave                         | 波浪/电流                        |             |
| 6    | Criss cross                  | 马步、交叉变换跃动               |             |
| 7    | Steve martin                 | 前走、后踢转身勾腿               |             |
| 8    | Jerk                         | 后踢×4、横摇×4                   |             |
| 9    | Cat daddy                    | 左右脚尖移动，侧身律动下蹲       |             |
| 10   | Kick cross step              | 前踢侧移                         |             |
| 11   | Walk it out                  | 旋脚尖前移                       |             |
| 12   | Dougie                       | 顶肩横移                         |    Urban         |
| 13   | Cabbag patch                 | 摇臂前后踩，三步平移             |             |
| 14   | Spongebob                    | 侧跃同向侧踢                     |             |
| 15   | Smurf                        | Rock+拳画圆                      |             |
| 16   | Shamrock                     | 横摇肩膀前后踩，三步平移         | Party Dance |
| 17   | Running man                  | 提腿踩地                         |             |
| 18   | Kick ball change             | 踢踩变换                         | Swag        |
| 19   | Happy feet                   | 脚跟脚尖点地，跃动变化(部分人把V step叫Happy feet)           |             |
| 20   | Roger rabbit                 | 后踢reject，重拍在up             |             |
| 21   | Heel toe                     | 脚跟脚尖                         |             |
| 22   | Funky chicken                | 左右摇跃+高度+横向手臂           |             |
| 23   | Funky broadway (crab, the v) | V字步                            |             |
| 24   | Crzy legs                    | walk it out，cross，turn，撤腿转身   | Poppin      |
| 25   | Run it                       | 前踩，脚尖V字旋                  |             |
| 26   | Body wave                    | wave头传至膝                     |             |
| 27   | Arm wave                     | 手臂十一步                       | Poppin      |
| 28   | Butterfly                    | 脚跟外扣                         |             |
| 29   | Stick and roll               | 单脚跟外扣，跟一个bodywave       |             |
| 30   | Pick it up and chuck it      | 右手下捞右扔，右手左扔，左手右扔 |             |
